# Kawaii FiveM Phone Theme Icons

Sevimli ve pastel emoji tarzında tasarlanmış 33 adet SVG uygulama ikonu koleksiyonu.

## Özellikler

- **33 Adet İkon**: FiveM telefon teması için özel tasarlanmış
- **SVG Format**: Ölçeklenebilir vektör grafikleri
- **Kawaii Stil**: Pastel renkler ve yumuşak tasarımlar
- **İnteraktif Galeri**: Yerel görüntüleyici ile kolay erişim
- **Tema Desteği**: Açık, koyu ve pastel tema seçenekleri
- **Arama**: İkonları isim veya kategori ile filtreleme
- **Kopyala/İndir**: SVG kodunu kopyalama ve indirme özelliği

## Kullanım

1. **Yerel Görüntüleyici**:
   ```bash
   # Klasöre git
   cd kawaii-fivetheme
   
   # Tarayıcıda aç
   # Windows: index.html dosyasına çift tıkla
   # Mac: open index.html
   ```

2. **FiveM'de Kullanım**:
   - SVG ikonlarını kullanmak istediğiniz uygulamanın assets klasörüne kopyalayın
   - CSS veya Lua kodunuzda ikon yolunu belirtin

3. **Web'de Kullanım**:
   ```html
   <img src="icons/phone.svg" alt="Phone">
   ```

## Tasarım Özellikleri

- **Renk Paleti**: Pastel tonlar (Pembe, Lavanta, Nane Yeşili, Şeftali)
- **Yuvarlatılmış Köşeler**: 16px border-radius
- **Gradient Arka Planlar**: Her ikon için özel gradient
- **Soft Gölgeler**: Yumuşak kutu gölgeleri
- **Kawaii Yüzler**: Minimalist emoji ifadeleri

## Teknik Detaylar

- **ViewBox**: 64x64
- **Format**: SVG 1.1
- **Renk Uzayı**: RGB
- **Gradient Türleri**: Linear gradient
- **Filtreler**: SVG drop-shadow
